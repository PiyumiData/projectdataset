*****Pre-requisites to run the project*****

1. Project source code files (folder names: data,features,models) submitted to Canvas should be saved in below path:
	C:\Project\src

2. Dataset (folder names:Augmented,Testing Images) which is on GitHub should be saved in below path:
	C:\Project\data

3. So, after saving the sourcecode folders and dataset folders in above paths:
	Training dataset will be in below path:
		C:\Project\data\Augmented\split\train
	Testing dataset will be in below path:
		C:\Project\data\Augmented\split\val
	GUI testing images are in below path
		C:\Project\data\Testing Images
	Saved models will be in below path:
		C:\Project\src\features
